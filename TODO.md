# 🚀 项目代号：Hire A Boss (自我重建计划)

> **核心理念**：我不是不想工作，我只是缺一个好老板。
> **目标**：打造一个能下达任务、给予奖励、明确方向的 AI 老板。

---

## 🎯 核心功能需求 (PRD)

### 1. AI 老板 (任务与鞭策)
- [ ] **任务生成**：每天早上 9 点，自动生成当日任务清单（工作技能 + 生活健康）。
- [ ] **定时鞭策**：如果任务未完成，下午 4 点发送提醒消息（语气要坚定但温和）。
- [ ] **任务验收**：完成任务后，需要向老板“汇报”，老板确认后才算完成。

### 2. 积分激励系统 (奖励)
- [ ] **赚取积分**：每完成一个任务 +10 分，每日会议 +5 分。
- [ ] **积分商城**：设定奖励列表（例如：100 分=休息半天，500 分=买个小礼物）。
- [ ] **兑换记录**：记录每一次兑换，防止滥用。

### 3. 每日会议 (方向对齐)
- [ ] **晨会/晚会入口**：提供一个输入框，用于写“今日发言”。
- [ ] **发言模板**：引导我写：昨天做了什么？今天打算做什么？遇到困难了吗？
- [ **老板反馈**：AI 根据我的发言，点评是否符合“公司长期发展方向”（即求职/成长目标）。

---

## 🛠️ 开发阶段规划 (MVP 路径)

### 第一阶段：原型验证 (本周)
*目标：不用写代码，先用工具跑通流程*
- [ ] 1. 使用 Notion 或 飞书多维表格 搭建任务列表。
- [ ] 2. 手动模拟"AI 老板”，每天早上给自己发 3 个任务。
- [ ] 3. 手工记录积分，验证奖励机制是否让自己有动力。
- [ ] 4. 每天写一段“会议发言”，自己复盘。

### 第二阶段：自动化实现 (下周开始)
*目标：引入 AI 工具，减少人工操作*
- [ ] 1. 使用 Replit Agent 或 Cursor 尝试生成网页版应用。
- [ ] 2. 接入大模型 API，让任务生成和会议反馈自动化。
- [ ] 3. 实现简单的积分累计逻辑。

---

## 🛡️ 老板的特别规定 (宪法)
- [ ] **禁止自我 PUA**：如果任务未完成，老板只能提醒，不能侮辱人格。
- [ ] **方向大于速度**：每日会议的重点是“方向是否正确”，而不是“做了多少事”。
- [ ] **健康优先**：连续工作 6 天，第 7 天强制休息。
